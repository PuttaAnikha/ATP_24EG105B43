# Week 2 - JavaScript Advanced

More JavaScript practice.
Covers array methods, objects, copying, and backend basics.

## What's in here

### 25-02-26 Folder

Assignments using array methods like filter, map, reduce, find, findIndex.

#### BankTransaction.js
Bank transactions - filter credits, map amounts, reduce balance, find debits.

#### EmployeePayroll.js
Employee payroll - filter IT dept, map net salary, reduce total payout, find by salary.

#### MovieStreaming.js
Movie platform - filter sci-fi, map titles with ratings, reduce average rating, find movies.

#### OnlineCourse.js
Online courses - similar array operations on course data.

#### ShoppingCart.js
Shopping cart - filter in-stock, map total prices, reduce grand total, find items.

#### StudentList.js
Student list - array methods on student data.

#### StudentPerformance.js
Student performance - operations on marks and grades.

#### ECommerceShoppingCart/
Mini e-commerce app with modules.

- App.js: Main app, browse products, cart operations.
- Cart.js: Cart functions - add, update, remove items.
- Discount.js: Discount calculations.
- Payment.js: Payment processing.
- Product.js: Product management.

#### TaskManagementSystem/
Task manager app.

- App.js: Main app, add and complete tasks.
- Task.js: Task functions.
- Validator.js: Input validation.

### 26-02-26 Folder

Hands-on exercises on copying and parameters.

#### DeepCopy.js
Deep copy using structuredClone, modify nested objects safely.

#### ShallowCopy.js
Shallow copy with spread, shows issues with nested changes.

#### SpreadAndRestParameter.js
Exercises on spread for copying arrays/objects, rest parameters.

#### ExamPortal.js
Exam portal - probably array/object operations.

#### LibraryManagement.js
Library system - manage books and users.

#### OTPCountdown.js
OTP countdown timer.

### backend Folder

Node.js backend with Express.

#### Server.js
Express server with middleware and routing.

#### API/ProductAPI.js
Product API routes.

#### API/UserAPI.js
User API routes.

#### package.json
Dependencies.

#### req.http
HTTP requests for testing.

## What you'll learn
- Array methods: filter, map, reduce, find, findIndex
- Shallow vs deep copy
- Spread and rest operators
- Modular code
- Basic backend with Express

## How to run
For JS files: node filename.js

For backend: npm install, then node server.js
