# Week 5: HTML/CSS and React Introduction

## Project Description

This week transitions from backend to frontend development, covering HTML/CSS styling and introduction to React framework. The assignments include styled web pages and basic React applications built with Vite.

## Features and Functionalities

- HTML table creation and styling
- CSS layout and design
- React component development
- Vite build system usage
- Component-based architecture
- Basic React state management

## Technologies Used

- **HTML5**: Web page structure
- **CSS3**: Styling and layout
- **React**: Component library
- **Vite**: Build tool and dev server
- **JavaScript (ES6+)**: Programming logic

## Installation Steps

For React applications:
```bash

```

## Steps to Run the Project Locally

### HTML/CSS Pages
Open .html files directly in browser or use local server.

### React Applications
```bash
cd week-5/18-03-26/react_application1
npm run dev
# Visit http://localhost:5173
```
