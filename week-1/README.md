Covers if-else, loops, arrays, functions, and objects.

#### day1.2.js

#### day1.3.js

#### day1.4.js
Finds the smallest number in an array by looping through it.

#### day1.5.js
A function that takes three numbers and returns the biggest one.

#### day1.6.js
A function that sums up an array of numbers.

#### day1.7.js
